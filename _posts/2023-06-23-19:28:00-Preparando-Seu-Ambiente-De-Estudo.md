# O Ambiente De Estudo

Compreendo que é difícil abranger todas as atividades de um especialista em cibersegurança em um único post. No entanto, neste artigo, farei o possível para mostrar algumas ferramentas básicas que utilizo e fornecer orientações sobre como configurá-las. É importante ressaltar que essas são apenas algumas das muitas ferramentas disponíveis, mas espero que essas informações sejam úteis para aqueles que desejam iniciar sua jornada na cibersegurança.

# Sistema Operacional

Neste post, abordarei o tema do "Kali Linux". No entanto, antes de prosseguir, é importante entender o que é o Linux. O Linux é um projeto de sistema operacional de código aberto, o que significa que seu código-fonte está disponível para qualquer pessoa modificar e desenvolver seu próprio sistema operacional baseado nele. Essa característica de código aberto resultou em uma variedade de distribuições Linux disponíveis, e uma delas é o Kali Linux, que será o foco deste post.

# Kali Linux

O "Kali Linux" é uma distribuição baseada no Linux Debian, projetada especificamente para profissionais de cibersegurança e entusiastas da área. É mantido atualizado pela empresa Offensive Security, que se dedica a trazer sempre as melhores ferramentas. Essa distribuição é reconhecida por sua abordagem voltada para a segurança e pela inclusão de várias ferramentas úteis para profissionais de segurança cibernética.

# Oracle VM VirtualBox

Para utilizarmos o Kali Linux, primeiro precisaremos baixar e instalar o "Oracle VM VirtualBox", que é um aplicativo que nos permite emular outros sistemas dentro do nosso próprio Windows. Essa ferramenta é essencial para criar um ambiente virtual seguro e isolado para executar o Kali Linux.

[Link Para Download](https://www.virtualbox.org/)

Ao acessar a página, você deverá clicar em "Download" localizado no canto esquerdo da página. Em seguida, clique em "Windows Hosts" para iniciar o download do instalador. Assim, você terá o arquivo de instalação do Oracle VM VirtualBox baixado em seu computador, Após o download do arquivo de instalação do Oracle VM VirtualBox, você pode executar o instalador. Dê um duplo clique no arquivo baixado para iniciar o processo de instalação. Siga as instruções na tela e clique em "Avançar" ou "Próximo" para prosseguir. Você pode selecionar as opções de instalação padrão ou personalizadas, conforme suas preferências.

# Instalando o Kali Linux

Primeiramente, devemos acessar a página do Kali Linux. Em seguida, clique na opção "Download" e escolha a categoria "Virtual Machines". Nesta pagina, você poderá encontrar diferentes opções de máquinas virtuais disponíveis para download. Selecione a opção "VirtualBox 64" para baixar a versão específica do Kali Linux que é compatível com o Oracle VM VirtualBox. Essa versão é projetada para funcionar corretamente com o VirtualBox e oferecer uma experiência otimizada.

[Kali Linux](https://www.kali.org/)

# Instalando O Kali Linux No Oracle VM

Para instalar o Kali Linux, devemos primeiramente extrair o arquivo. Após extraído, você deverá abrir o Oracle VM e clicar em "Acrescentar". Em seguida, navegue até a pasta onde o Kali Linux foi baixado e abra o arquivo que está presente nessa pasta, vale lembrar que o usuario e senha padrão é : kali,kali
